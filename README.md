## Getting Started

To get a local copy of the repository please run the following commands on your terminal:

$ cd  folder-name

~~~bash
git clone git@github.com:uchennaanya/members_only.git
$ cd members_only
$ bundle install 
$ rails db:migrate
$ yarn install --check-files
~~~
